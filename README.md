# Paradise Nursery

Paradise Nursery is an online houseplant shopping application built with React and Redux.

## Features

- Beautiful landing page
- About Us section
- Product listing page
- Three categories of plants
- Shopping cart
- Redux state management
- Dynamic cart count
- Increase/decrease quantity
- Remove items
- Checkout button

## Technologies

- React
- Redux Toolkit
- React Router
- CSS
